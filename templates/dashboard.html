About Threat Feeds

Threat intelligence feeds provide real-time information about potential security threats from various sources around the internet.

Feed Sources:

  • ThreatFox - Malware IOC database
  • PhishTank - Community-verified phishing URLs
  • URLhaus - Database of malicious URLs
  • Feodo Tracker - Botnet C2 servers
  • CISA Known Exploited Vulnerabilities

The platform automatically collects and processes these feeds to extract actionable intelligence for your security operations.

IOC Information

Indicators of Compromise (IOCs) are artifacts observed in networks or systems that indicate potential security incidents or malicious activity.

Common IOC Types:

IP Malicious IP addresses
Domain Malicious domains
URL Malicious URLs
Hash File hashes (MD5, SHA1, SHA256)

IOCs are extracted automatically from threat feeds and can be used to detect potential threats in your environment.
